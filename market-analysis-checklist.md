# AI-анализ рынка — чеклист для человека

Используй параллельно с AI-агентом, который ведёт анализ рынка.  
Эти шаги **ты** выполняешь сам — AI не может сделать их за тебя.

---

## Перед стартом: форма для фрейминга

Заполни **до** того, как ставить задачу AI. Незаполненные поля = AI угадает, и угадает неправильно.

```
Продукт/сервис: ___
Целевой клиент (сегмент): ___
География: ___
Временной горизонт: ___ (год / период)
Метод расчёта: [ ] top-down  [ ] bottom-up  [ ] value theory  [ ] все три
Цель оценки:   [ ] питч  [ ] стратегия  [ ] go/no-go решение
```

---

## После того как AI вернул результаты: красные флаги

Останови и перепроверь вручную, если **хотя бы одно** из ниже — правда:

- Нет ссылок на источники → скорее всего выдумка
- Все цифры круглые (100 млрд, 50%, 10 млн) → нет реальных данных
- «По данным отраслевых исследований» без названия → классическая галлюцинация
- Нет конкретных игроков рынка → AI не нашёл данные
- Нет диапазона / uncertainty → AI притворяется, что точно знает
- Данные 2021–2022 для РФ-рынка → мир изменился после этого
- SOM занимает >30% SAM с первого года → нереалистично почти никогда
- «Рынок вырос в 3× за год» без объяснения → AI экстраполировал, а не нашёл
- Ссылка на Statista / Grand View / MarketsandMarkets → платные данные, которых AI не видел

---

## Проверка порядка величин (отвечаешь ты, не AI)

- SOM реален при нашем размере команды и горизонте?
- Проникновение реалистично (≤30% в год для нового рынка — уже агрессивно)?
- Темп роста рынка объяснён реальным событием или AI просто экстраполировал?

---

## Частые ошибки (по реальным кейсам)

1. **Забыли указать географию** → AI посчитал мировой рынок, а не РФ
2. **Не задали метод** → AI сделал top-down, хотя есть внутренние данные для bottom-up
3. **Не запустили adversarial review** → красивые цифры ушли в питч-дек
4. **Не учли сезонность** → SOM в годовой выручке, но бизнес сезонный
5. **Adoption нового канала** → не учли долю нового канала в общем объёме рынка
6. **Рынок мал, но идея нравится** → натянули цифры; надо вернуться к стратегическому фильтру

## Формула (напоминание)

```
Потенциал = Рынок × Доля/Выручка × Активы × Вера в успех
            ─────────────────────────────────────────────────
                  Сложность (Время + Люди + Деньги)
```

AI закрывает **Рынок** и частично **Активы**.  
**Вера в успех** (стратегическое попадание) — твоё решение.  
**Сложность** — внутренняя оценка команды, AI не знает твоих ограничений.
